# C++ 변수 초기화 방법

```c++
#include <iostream>
int main()
{
	int x = 10; //copy initialization 복사 초기화
	int y(2);//direct initialization 직접 초기화
	int z{ 3 };//Uniform initialization, C++11 유니폼 초기화
	int z1{};//Uniform initialization, 자동으로 0,C++11
	std::cout << x << y << z << z1;
}

```
