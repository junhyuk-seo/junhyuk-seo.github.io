# Namespace

모든 식별자가 유일하도록 코드 영역을 정의합니다.
Namespace가 없으면 식별자가 동일할 경우 혼란이 옵니다. 

## std namespace
C++ 표준 라이브러리의 네임스페이스(cin, cout등)

# Using 지시문

using namespace std;
std 네임스페이스의 모든 걸 사용하고 싶다는 의미입니다.
컴파일러가 네임스페이스가 없는 식별자를 만나면 std 네임스페이스에서 찾습니다.
